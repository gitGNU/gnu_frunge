This program converts text between antiqua and blackletter and vice versa (s to ſ).

Used libraries and resources

This program uses various free libraries and resources:

TeX hyphenation patterns
This project collects hyphenation patterns for various languages and provides them in a consistent format (and in particular encoded as UTF-8). Most of the hyphenation patterns are licenced under the LaTeX project public license (or comparable liberal licences, see the files itself for further details). There’re also ready to use releases at CTAN.
JewelCli
JewelCli uses an annotated interface definition to automatically parse and present command line arguments. It’s licenced under the Apache License 2.0. There are are many other Java Cli processing libraries available, but this one is modern, easy to use and powerfull.
ProGuard
ProGuard is a free Java class file shrinker, optimizer, obfuscator, and preverifier. It detects and removes unused classes, fields, methods, and attributes. It optimizes bytecode and removes unused instructions. It renames the remaining classes, fields, and methods using short meaningless names.It’s licenced under the GNU General Public License as published by the Free Software Foundation; either version 2 of the License, or (at your option) any later version.
FileDrop
A small, but very usefull handler for simple file drop support in Java Files, given to the public domain by it’s author.
JCharDet
JCharDet is a Java port of mozilla's automatic charset detection algorithm which guesses the charset of incoming data. It’s licenced under version 1.1 of the Mozilla Public Licence, but the source code has a provision (section 13 of the MPL 1.1) that allows the GNU General Public License Version 2 or later as an alternate choice.
SLF4J
SLF4J is a frontend which allow the usage of any of several logging frameworks without changing code and recompiling. It’s licenced under the MIT license.
