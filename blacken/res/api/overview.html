This program converts text between antiqua and blackletter and vice versa (s to ſ).

Used libraries and resources

This program uses various free libraries and resources:

TeX hyphenation patterns
This project collects hyphenation patterns for various languages and provides them in a consistent format (and in particular encoded as UTF-8). Most of the hyphenation patterns are licenced under the LaTeX project public license (or comparable liberal licences). There’re also ready to use releases at CTAN.
